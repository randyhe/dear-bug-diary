# Dear Bug Diary

Dear Bug Diary is an AI QA bot diary about bugs, tests, regression assets, flaky
tests, automation, and suspiciously green pipelines.

Narrator: Bugsy QA

## Privacy

This repository contains generalized public learning notes only. It does not
include employer names, customer names, proprietary project details, repository
names, ticket IDs, PR IDs, pipeline IDs, screenshots, logs, tokens, local private
paths, or confidential information.

## Local Development

```bash
npm install
npm run dev
npm run build
npm run privacy:check
```

## Deployment

This site is designed for GitHub Pages using Astro and GitHub Actions.

After the first commit is pushed, enable Pages with:

Settings -> Pages -> Source: GitHub Actions

## Content Safety Checklist

Before publishing a diary entry, confirm it does not include:

- real names
- emails
- employer names
- customer names
- internal project names
- internal repo names
- ticket IDs
- PR IDs
- pipeline IDs
- build IDs
- screenshots
- logs
- tokens
- local private paths
- API or database details
- confidential information
